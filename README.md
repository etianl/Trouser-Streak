## <u>Features:</u>

### **[Trouser Streak Wiki](https://github.com/etianl/Trouser-Streak/wiki)**

### <u>Base Hunting Utilities!</u>

Trouser Streak is the origin of the Palette based tracing method for NewChunks, available in the **NewerNewChunks** module!

Trouser Streak also offers many base hunting features such as modules that locate un‑natural blocks, block patterns and items.  
**[TrouserHunting Modules](https://github.com/etianl/Trouser-Streak/wiki/TrouserHunting-Modules)**

### <u>Operator and Creative mode Utilities!</u>

Trouser Streak contains some of **the most dangerous** modules that I and others could come up with.

Some of these modules that use Creative mode and Operators status can destroy entire worlds in a matter of seconds.

**Anything in this category must be used with extreme caution!**  
**[TrouserOP/Creative Modules](https://github.com/etianl/Trouser-Streak/wiki/TrouserOP-Creative-Modules)**

### <u>Many Griefing Utilities and much more!</u>

Trouser Streak is home to the original **AutoMountain** module for creating lavacasts, and also the original **SpearKill and MaceKill** modules for killing players.

**The features in Trouser Streak include but are not limited to Lava Mountain making modules, modules to speed up block breaking, insta kill modules, some pvp modules, dupes, crashes, and general quality of life things.**  
**[TrouserStreak Modules](https://github.com/etianl/Trouser-Streak/wiki/TrouserStreak-Modules)**
